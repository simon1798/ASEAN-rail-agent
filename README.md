## 📋 项目概述

东盟轨道交通研究Agent是一个专业的智能决策支持系统，为中国轨道交通企业出海东盟市场提供全面的市场情报、竞争分析、风险评估和机会匹配服务。

### 核心价值

- **市场监测**: 实时追踪东盟10国轨道交通项目动态
- **竞争情报**: 分析中国企业出海能力与国际竞争对手
- **风险预警**: 基于历史案例的智能风险评估系统
- **机会匹配**: AI驱动的项目-企业智能匹配引擎
- **决策支持**: 数据驱动的市场进入策略建议

## 🏗️ 系统架构

### 技术栈

```
核心引擎: Python 3.10+
      ├─ LangChain (多Agent协作)
      ├─ Pandas (数据处理)
      └─ Plotly (数据可视化)

数据层:
      ├─ PostgreSQL (结构化数据)
      ├─ ChromaDB (向量知识库)
      └─ Redis (实时缓存)

前端: Streamlit 1.32+
爬虫: Scrapy + Playwright
调度: Celery + Redis
```

│   │   ├── 风险案例库 (4类风险)
│   │   └── 数据源库 (招标平台)
│   └── 分析引擎
│       ├── 市场机会分析
│       ├── 竞争格局分析
│       ├── 风险评估

asean_rail_dashboard.py (可视化界面)

## ⚠️ 风险预警系统

### 高风险案例

**1. 菲律宾北吕宋铁路（政治风险）**
- 项目周期：2003-2017年（14年拉锯）
- 最终结果：和解仅获6270万美元
- 关键教训：政权更迭、合同有效性

**2. 沙特麦加轻轨（低价中标）**
- 亏损：41.53亿元
- 根本原因：工程量严重低估、标准不熟悉
- 关键教训：拒绝恶性竞价

**3. 南非Transnet（法律合规）**
- 资金冻结：21亿元
- 关键问题：税务合规、外汇管制
- 关键教训：合规审查

### 风险评估指标

- **政治风险**: 政权更迭、政策变动
- **财务风险**: 汇率波动、资金冻结
- **法律风险**: 合同有效性、合规审查
- **技术风险**: 标准差异、工程量估算
